Instrucciones:
    - El archivo "laberinto.txt" debe estar en la misma
    ruta que los archivos .java
    - Para usar el programa, se debe estar en el directorio donde estan los archivos
    y poner "make run1", para el caso de hebras y "make run2" para el caso de forks.
    - Para borrar los .class se puede usar "make clean".
    - Se uso javac 11.0.19 para compilar.

Archivos necesarios:
    - Funciones.java
    - BaquedanoForks.java
    - BaquedanoThread.java
    - MiRunnable.java
    - laberinto.txt
    - Makefile
